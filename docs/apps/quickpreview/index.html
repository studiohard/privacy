About QuickPreview

QuickPreview is a native macOS app with Quick Look extensions. It helps you inspect folder contents, Markdown documents, and archive contents more quickly without opening them in a full editor or Finder window.

Key Features

  • Preview folder contents directly in Quick Look
  • Preview Markdown files with a richer reading experience
  • Inspect supported archive contents before extracting
  • Customize preview display settings such as layout and visibility options
  • Privacy-first design with local-only processing
