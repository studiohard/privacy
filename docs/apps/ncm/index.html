NCM 音频转换器

NCM 音频转换器是一款 macOS 本地工具，用于将用户主动选择的本地 .ncm 文件转换为 MP3 文件。

支持网站 隐私协议

关于应用

应用专注于离线转换场景。你选择文件、选择输出目录，转换结果会保存在你的 Mac 上。

核心功能

  • 导入本地 .ncm 文件
  • 转换为通用 MP3 格式
  • 自定义输出目录
  • 查看转换历史并在 Finder 中定位结果
  • 本地处理，不上传音频文件

快速链接

  • 支持网站 - 常见问题和联系方式
  • 隐私协议 - 应用如何处理你的数据
